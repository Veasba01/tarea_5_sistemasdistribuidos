
## Instrucciones de la Tarea

1. **Cree una cuenta en Redis**
   - Regístrese en: https://redis.com  
   - Agregue una base de datos llamada **book-store**.

2. **Autores – Backend y Frontend**
   - Desarrolle funciones backend y componentes frontend para:
     - Listar autores  
     - Agregar autores  
     - Modificar autores  
     - Eliminar autores  
   - Toda la información debe almacenarse en Redis.

3. **Editoriales – Backend y Frontend**
   - Programe las funciones backend y los componentes frontend para:
     - Listar editoriales  
     - Agregar editoriales  
     - Modificar editoriales  
     - Eliminar editoriales  
   - La información debe administrarse usando Redis.

4. **Publicación de la Aplicación**
   - Publique el backend y frontend en **Netlify**.
